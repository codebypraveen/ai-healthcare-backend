# 🩺 AI Healthcare Disease Prediction System

An AI-powered healthcare system that predicts possible diseases based on user symptoms and recommends medicines.

This project uses Machine Learning to analyze symptoms and generate a medical diagnostic report.

---

## 🚀 Live Demo

Frontend:  
https://medicarepro2.netlify.app/
Backend API:  
https://ai-healthcare-backend-ujw1.onrender.com/

---

## 🧠 Features

- Symptom-based disease prediction
- Top 3 possible disease suggestions
- Confidence level for each prediction
- Recommended medicines
- Emergency alert for serious diseases
- AI diagnostic report generation
- PDF report download
- Searchable symptom selection
- Cloud deployment

---

## 🏗 System Architecture

Frontend → Netlify  
Backend API → Render  
Machine Learning Model → Python (Scikit-learn)

Workflow:

User Inputs Symptoms  
↓  
Frontend sends request to API  
↓  
Flask backend processes data  
↓  
Machine Learning model predicts disease  
↓  
System generates AI Healthcare Diagnostic Report
